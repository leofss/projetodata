## Instruções para Implementação Com Docker

### Passos para Configuração

1. Clone o repositório:   
    ```bash
    git clone https://github.com/leofss/projetodata.git
    cd projetodata/backend
    ```
2. Suba o container (**verifique se a porta 8080 e 3000 estão disponíveis**)
   ```bash
    docker compose up --build
    ```
3. Acesse [Localhost](http://localhost:3000) para visualizar o frontend
